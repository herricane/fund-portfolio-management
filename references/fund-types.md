# 中国公募基金类型速查（给非专业投资者）

> 目的：当用户问「纯债收益低正常吗」「红利为什么亏」时，快速给出有依据的答案。

## 📜 纯债基金

| 维度 | 说明 |
|------|------|
| 投什么 | 国债、政策性金融债、高等级信用债 |
| 怎么赚钱 | 票息收入 + 利率下行时债券价格上涨 |
| 预期年化 | 2.5-4% |
| 最大回撤 | <1%（纯短债）、<3%（中长期纯债） |
| 亏损原因 | 利率大幅上行（如 2022 年 11 月债灾） |
| 对组合的价值 | 压舱石：市场暴跌时它不跌甚至涨（避险资金涌入国债） |
| 正常吗 | 年化 3%、几乎不回撤 → 完全正常，这就是它该有的样子 |
| 异常信号 | 单日跌 >0.5%、连续周线下跌 → 利率环境剧变 |

## 💰 红利基金（中证红利）

| 维度 | 说明 |
|------|------|
| 投什么 | A 股中股息率最高的 100 只股票（银行、煤炭、钢铁、公用事业为主） |
| 怎么赚钱 | 股息分红 + 股价上涨 |
| 预期年化 | 8-12%（长期） |
| 最大回撤 | 15-25% |
| 亏损原因 | 市场风格偏向成长/科技时，资金从「旧经济」撤出；或煤炭/钢铁周期下行 |
| 对组合的价值 | 防御：波动低于宽基，股息提供稳定现金流 |
| 正常吗 | 在 AI/半导体牛市里红利亏 5% → 正常，是风格轮动的代价 |
| 异常信号 | 红利跑输宽基超过 20 个百分点持续 1 年以上 |

## 🏛 宽基指数（沪深300 / 中证500）

| 维度 | 说明 |
|------|------|
| 投什么 | 沪深300：A股市值最大300只；中证500：第301-800大 |
| 怎么赚钱 | 指数上涨 + 分红再投资 |
| 预期年化 | 8-12%（长期，含分红） |
| 最大回撤 | 25-40% |
| 对组合的价值 | 基底：整个组合的地基，不挑赛道不择时 |

## 💻 行业/主题基金（科技、半导体）

| 维度 | 说明 |
|------|------|
| 投什么 | 特定行业的股票，集中度高 |
| 怎么赚钱 | 行业景气上行 + 个股上涨 |
| 预期年化 | 10-25%（高波动、高回报） |
| 最大回撤 | 30-50%（行业下行时极其惨烈） |
| 对组合的价值 | 卫星：押注看好的赛道，不应成为核心仓位 |
| 异常信号 | 单行业 >30% 仓位、月定投超 40% 流入 → 风格漂移 |

## 🇺🇸 QDII 基金（纳指、标普）

| 维度 | 说明 |
|------|------|
| 投什么 | 海外（主要是美股）指数 |
| 怎么赚钱 | 美股上涨 + 人民币贬值（QDII以人民币计价） |
| 预期年化 | 10-15%（纳指长期） |
| 最大回撤 | 25-40% |
| 特殊限制 | QDII 额度有限，常限购（10-200元/天） |
| 对组合的价值 | 地理分散：A股和美股不完全同步 |

## 🛢 商品基金（黄金、资源）

| 维度 | 说明 |
|------|------|
| 投什么 | 黄金ETF跟踪金价；资源基金：能源、有色矿企股票 |
| 怎么赚钱 | 金价上涨、资源品价格上涨 |
| 预期年化 | 黄金 3-6%（长期）、资源 6-10% |
| 最大回撤 | 黄金 20-30%、资源 30-50% |
| 对组合的价值 | 通胀对冲 + 极端事件保险（金价在危机中往往上涨） |
| 正常吗 | 黄金亏 11% 在和平/利率高位环境 → 正常。它的价值在危机中才体现 |
