# 中国公募基金类型速查

> 目的：当用户问「纯债收益低正常吗」「红利为什么亏」「这只基金属于哪类」时，快速给出有依据的答案。
> 覆盖中国公募基金主要类型，而非仅作者个人持仓。

---

## 投资逻辑速查（补仓/止损判断用）

判定单仓亏损 > 10% 时「逻辑是否还在」，查此表：

| 资产类别 | 投资逻辑 | 什么算「逻辑破了」 |
|---------|---------|-----------------|
| 宽基 | 中国经济长期增长，大盘指数跟随上行 | 系统性金融危机、持续通缩、资本管制 |
| 海外 QDII | 地理分散 + 全球市场增长 | QDII 制度取消、美股长期结构性衰退 |
| 纯债/利率债 | 利息收入 + 利率下行资本利得 + 危机避险 | 连续加息、大规模违约、恶性通胀 |
| 可转债/混合债 | 债底保底 + 权益弹性 | 股市长期熊市 + 利率上行（双重打击） |
| 红利/低波 | 高股息 + 低波动，熊市防御 | 企业系统性削减分红、红利税重大变更 |
| 行业/主题 | 特定赛道景气上行 | 行业颠覆性监管、技术替代、需求永久萎缩 |
| 黄金 | 通胀对冲 + 极端事件保险 | 全球回归极低通胀（几乎不可能） |
| 商品/资源 | 通胀对冲 + 供需周期 | 需求结构永久变化、替代技术颠覆 |
| 货币基金 | 流动性管理，几乎无风险 | 大规模赎回潮（极端罕见） |

> **原则**：宽基/纯债/货币的「逻辑」很少会破。行业主题和商品的「逻辑」最可能破。
> 「逻辑破了」不等于必须清仓，而是**停止补仓 + 考虑减配**。

## 一、按资产类别

### 📜 货币基金

| 维度 | 说明 |
|------|------|
| 投什么 | 短期国债、央行票据、银行存单、高等级短融 |
| 怎么赚钱 | 票息收入，几乎无价格波动 |
| 预期年化 | 1.5-2.5% |
| 最大回撤 | ≈0%（极罕见情况下单日负收益） |
| 对组合的价值 | 现金替代：流动性管理、定投资金暂存、应急储备 |
| 典型代表 | 余额宝、零钱通背后的货币基金 |

### 📜 纯债基金

| 维度 | 短债/中短债 | 中长期纯债 |
|------|-----------|-----------|
| 投什么 | 剩余期限 <1-3 年的债券 | 无期限限制，以中长期为主 |
| 怎么赚钱 | 票息为主，价格波动极小 | 票息 + 利率下行时价格上涨 |
| 预期年化 | 2-3.5% | 3-5% |
| 最大回撤 | <0.5% | 1-3% |
| 亏损原因 | —（几乎不回撤） | 利率大幅上行（如 2022.11 债灾） |
| 对组合的价值 | 短期理财替代 | 压舱石：股市暴跌时它不跌甚至涨 |

**异常信号**：单日跌 >0.5%、连续周线下跌 → 利率环境剧变。

### 📜 利率债/政策性金融债基金

| 维度 | 说明 |
|------|------|
| 投什么 | 国债、国开债、农发债等利率债（无信用风险） |
| 怎么赚钱 | 利率下行 + 骑乘效应（长久期债券 roll-down） |
| 预期年化 | 3-5% |
| 最大回撤 | 2-5%（久期越长越大） |
| 与纯债的区别 | 只投利率债不投信用债，信用风险为零，但利率敏感度高 |

### 🔄 可转债基金

| 维度 | 说明 |
|------|------|
| 投什么 | 可转换债券（可转股的债券，兼具债性和股性） |
| 怎么赚钱 | 债底保底 + 正股上涨时转股获利 |
| 预期年化 | 5-15%（波动很大） |
| 最大回撤 | 15-30% |
| 特点 | 涨时像股票、跌时像债券（但不完全——极端下跌时转债也大跌） |
| 对组合的价值 | 另类资产，提供不对称收益结构 |
| 适合谁 | 想参与股市但不愿直接买股票的保守投资者 |

**注意**：可转债基金 ≠ 纯债。风险远高于纯债。不要在组合中把它当作"债券"对待。

### 🔄 混合债（一级债/二级债）

| 维度 | 一级债基 | 二级债基 |
|------|---------|---------|
| 投什么 | 纯债 + 可转债（不可直接买股票） | 纯债 + ≤20% 股票 |
| 预期年化 | 4-7% | 5-8% |
| 最大回撤 | 3-8% | 5-12% |
| 对组合的价值 | 比纯债略高的回报 | 股债混合，自带再平衡 |

## 二、按投资策略

### 🏛 宽基指数

| 维度 | 沪深 300 | 中证 500 | 中证 A500 | 创业板指 | 科创 50 |
|------|---------|---------|----------|---------|--------|
| 覆盖 | 市值最大 300 | 第 301-800 大 | 行业均衡 500 | 创业板精选 | 科创板精选 |
| 风格 | 大盘价值 | 中盘成长 | 大盘均衡 | 成长 | 硬科技 |
| 预期年化 | 8-10% | 10-12% | 8-10% | 12-18% | 15-25% |
| 最大回撤 | 25-35% | 30-40% | 25-35% | 35-50% | 40-55% |
| 对组合的价值 | 地基 | 中小盘补充 | 新一代核心 | 成长侧重 | 科技侧重 |

**其他重要宽基**：上证 50（超大盘）、中证 1000（小盘）、国证 2000（微盘）。

### 🎯 策略指数（Smart Beta）

| 策略 | 代表指数 | 特点 | 预期年化 | 最大回撤 |
|------|---------|------|---------|---------|
| 红利 | 中证红利 | 高股息，低波动 | 8-12% | 15-25% |
| 低波动 | 中证 500 低波 | 选低波动个股 | 8-12% | 20-30% |
| 价值 | 沪深 300 价值 | 低估值因子 | 8-12% | 25-35% |
| 成长 | 沪深 300 成长 | 高增长因子 | 10-15% | 30-40% |
| 质量 | 中证质量 | 高 ROE 因子 | 10-15% | 25-35% |

**对组合的价值**：Smart Beta 在特定市场环境下可能跑赢普通宽基，但也会跑输。红利策略在防御型组合中尤其有价值（低波动 + 股息收入）。

### 📊 指数增强基金

| 维度 | 说明 |
|------|------|
| 投什么 | 跟踪指数，但通过量化模型小幅偏离以获取超额 |
| 目标 | 年化跑赢基准 2-5% |
| 费用 | 管理费略高于纯被动基金（0.5-1.0% vs 0.15-0.5%） |
| 风险 | 增强策略可能失效，跑输基准 |
| 对组合的价值 | 在指数收益基础上争取一点增强 |

## 五、另类/商品

### 🥇 黄金

| 维度 | 说明 |
|------|------|
| 投什么 | 黄金现货合约（Au99.99），跟踪国际金价 |
| 怎么赚钱 | 金价上涨 |
| 预期年化 | 3-6%（长期） |
| 最大回撤 | 20-30% |
| 对组合的价值 | 通胀对冲 + 极端事件保险（危机中往往上涨） |
| 正常表现 | 和平/利率高位环境中亏损 10-15% → 正常。它的价值在危机中体现 |

### 🛢 商品/资源

| 子赛道 | 说明 | 预期年化 | 最大回撤 |
|--------|------|---------|---------|
| 原油/能源 | 跟踪油价 | 5-15% | 40-60% |
| 有色金属 | 铜铝锂等矿企 | 8-15% | 35-50% |
| 资源综合 | 能源 + 有色 + 材料 | 6-12% | 30-45% |

**特点**：商品基金提供通胀对冲和分散化价值，但**不是长期增长资产**——商品本身不产生现金流。

### 🏢 REITs / 不动产

| 维度 | 说明 |
|------|------|
| 投什么 | 基础设施 / 商业地产 REITs |
| 预期年化 | 5-10% |
| 对组合的价值 | 实物资产敞口，提供租金收入流 |
| 中国现状 | 公募 REITs 品类有限，以基建/产业园为主 |

## 六、混合型（按股债比例）

| 类型 | 股票比例 | 预期年化 | 最大回撤 | 适合 |
|------|---------|---------|---------|------|
| 偏股混合 | 60-95% | 8-12% | 25-40% | 进取型 |
| 平衡混合 | 40-60% | 6-9% | 15-25% | 稳健型 |
| 偏债混合 | 0-40% | 4-7% | 5-15% | 保守型 |
| 灵活配置 | 0-95%（灵活调整） | 不确定 | 不确定 | 不想自己调仓的 |

**注意**：灵活配置型基金的实际仓位取决于基金经理判断，可能偏离你的组合目标。如果自己正在做核心-卫星管理，纯被动指数基金比灵活配置型更可控。

## C 类 vs. A 类费率参考

| | C 类 | A 类 |
|------|------|------|
| 申购费 | ¥0 | 0.08-0.15%（第三方平台折扣后） |
| 管理费 | 0.5-1.0%/年 | 0.15-0.5%/年 |
| 赎回费 | ≥30 天 ¥0 | 阶梯递减（通常 >2 年 ¥0） |
| 综合成本（持有 1 年） | 略高 | 略低 |
| 综合成本（持有 >2 年） | 显著更高 | 显著更低 |
| 适合 | 定投 / 持有 <2 年 | 大额一次投入 / 长持 |
