# 斯文森/耶鲁模型 — 资产配置框架

> 来源：David Swensen、《Unconventional Success》《Pioneering Portfolio Management》、Bloomberg/FT 报道

## 耶鲁捐赠基金六类资产

| 资产类别 | 目标权重 | 说明 |
|---------|---------|------|
| 国内股票 | ~2-5% | 大幅降低，通过 PE 间接持有 |
| 国外股票 | ~10-15% | 发达+新兴市场 |
| 债券 | ~5-10% | 国债/TIPS，流动性管理 |
| 实物资产 | ~20-25% | REITs、自然资源、基础设施 |
| 私募股权 | ~30-35% | VC + Buyout，核心超额来源 |
| 绝对收益 | ~20-25% | 对冲基金策略，低相关正回报 |

## 个人投资者简化版（《不落俗套的成功》）

| 资产类别 | 权重 | 中国替代品 |
|---------|------|-----------|
| 国内股票 | 20% | 沪深300/中证A500 |
| 发达市场 | 15% | 标普500 QDII |
| 新兴市场 | 5% | A股本身即新兴市场 + 港股通 |
| TIPS/国债 | 15% | 国债/利率债指数基金 |
| 长期国债 | 15% | 国开债/政策性金融债 |
| REITs | 20% | 基础设施REITs + 黄金ETF |
| 商品 | 10% | 资源+黄金（控制在15%以内） |

## 再平衡纪律

- 每半年或每年一次
- 偏离 >5-10% 触发
- 逆势操作：卖出强势资产，买入弱势资产
- 优先用新增资金，避免卖出

## 中国适配注意事项

1. 无真正公募 REITs — 用黄金+资源替代部分
2. 无公募 PE — 科创50+创业板部分替代
3. QDII 额度受限
4. 无资本利得税，但 30 天内赎回费高
5. 再平衡优先用新增资金

## 三大核心精神

1. 真正意义上的分散化（跨传统股债边界）
2. 拥抱非流动性溢价
3. 成本意识与被动投资
