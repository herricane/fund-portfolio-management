# 基金定投相关的认知偏误与纠正策略

> 来源：Kahneman & Tversky (1979)、Thaler、定投场景适配

## 七大偏误速查

| 偏误 | 定投场景 | 纠正规则 |
|------|---------|---------|
| **损失厌恶** | 下跌暂停定投、上涨加量 | 自动化扣款隔离情绪；下跌=更多份额 |
| **锚定效应** | 以买入成本为锚拒绝止盈 | 用估值百分位替代成本价决策 |
| **处置效应** | 小赚就跑、亏损死扛 | 卖出规则绑定逻辑而非盈亏 |
| **心理账户** | 区别对待不同来源资金 | 只看总资产组合 |
| **框架效应** | 浮亏 vs 实际亏损认知偏差 | 问"如果是现金，现在买吗？" |
| **追涨杀跌** | 连涨加仓、大跌赎回 | 冲动操作强制等 7 交易日 |
| **羊群效应** | 热销基金跟风 | 热销慎投、冷却 48 小时 |

## 追涨杀跌触发条件

| 触发 | 心理机制 | 定投例子 |
|------|---------|---------|
| 连涨 3 阳线 | 确认偏误+羊群 | 暂停债基转投科技 |
| 朋友圈热议 | FOMO | 加仓被讨论基金 |
| 指数新高 | 锚定+过度自信 | 加倍定投金额 |
| 连跌 >20% | 损失厌恶+恐慌 | 当晚提交赎回 |
| 政策冲击 | 可得性启发 | 立刻赎回相关基金 |
| 回本瞬间 | 锚定+处置 | 刚回本就卖，错过翻倍 |

## 纠正清单

1. 下跌只加不减，上涨不加量
2. 用估值百分位替代买入成本
3. 盈利卖一半，亏损先问逻辑
4. 按总资产百分比看问题
5. 每次问"现金此时买不买"
6. 冲动等 7 个交易日
7. 热销慎投，冷门关注
